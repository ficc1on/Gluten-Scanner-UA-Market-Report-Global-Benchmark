# Gluten Scanner Ukraine Market Report

Професійний консалтинговий звіт про потенціал продукту **«Глютеновий сканер»** для ринку України.

У цьому звіті «глютеновий сканер» — це цифровий продукт: мобільний застосунок, PWA або сайт для перевірки харчового товару за штрихкодом, OCR складу або пошуком. Це не фізичний хімічний тест на глютен.

## Мета

Оцінити, чи має цифровий gluten scanner потенціал як продукт для українського ринку, які гіпотези треба перевірити через MVP, і за яких умов продукт може стати нішевим self-sustaining бізнесом або beachhead для ширшого allergy scanner.

## Ключова логіка

Звіт відділяє факти від гіпотез:

- глобальний gluten-free market використано як tailwind, не як direct TAM;
- Україна змодельована сценарно через нестачу актуальної офіційної статистики по целіакії;
- фінансова модель включає MAU, paid conversion, ARPPU, OCR COGS, platform fee, ФОП 5%, CAC/LTV і B2B upside;
- pricing для України позначений як WTP hypothesis і має перевірятися через MVP;
- primary user insight базується на lived experience автора як людини з целіакією в Україні, але не замінює MVP-валідацію.
